## 2025-05-16 - [Improved contrast for footnote text]
**Learning:** Hardcoded hex colors like #bbbbc2 fail WCAG AA contrast standards (2:1 ratio). Using #6b7280 improves this to ~4.5:1.
**Action:** Always check contrast ratios for subdued text in UI components.
